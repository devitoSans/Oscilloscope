# Status
On-going project. A lot of features have not been implemented.

# About
A project to build an oscilloscope capable of measuring up to 10KHz signal.

# Features
### Measurement Capabilities
1. Up to 10 KHz signal (Current or Voltage)
2. Measuring resistance
3. High signal protection
### User Interface
1. Dynamically visualise the measured signal
2. Trigger level feature
3. Apply FFT
